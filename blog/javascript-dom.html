AN ANALOGY TO DESCRIBE JAVASCRIPT AND IT'S RELATIONSHIP TO HTML AND CSS

People often compare the relationship between JavaScript, HTML and CSS all provide the basis to building a functional website, people compare the use of these 3 tools in conjunction to the process of putting together a house. HTML we use to describe the content of the website, or with a house this would be the bare essentails such as structure, doors and rooms. CSS is what we use to style the website (colours, typograhy etc) as you can imagine with a house this would be the paint, the size of a table, chair or doorway. Lastly we have JavaScript which allows users to interact with the website - for example, we would use Javascript to make sure that when we click on a "Make Booking" at a restaurant we would be able to make a reservation. If we were able to compare this to a house, Java would make sure the lights turn on or the water comes out of the tap. In conclusion all three of these counterparts must work together in order for the website to maximise the potential of the modern web.


EXPLAIN CONTROL FLOW AND LOOPS USING AN EXAMPLE PROCESS FROM EVERYDAY LIFE


We use loops in JavaScript when we want to execute a piece of code over and over again. There are three types of loops in JavaScript a FOR loop, a WHILE loop and do ... while loop. An everyday life example of when you may interact with a loop would be when you listen to a playlist. While you have another song that's availble to be played from the playist, the loop will keep going through your song list and stop once there is no more music to be played.


DESCRIBE WHAT THE DOM IS


If you were to create a document in Word, the document will be split into numerous sections, these could include a header, a title, a footer, and some paragraphs. There could also be images and graphs included in the document too. The DOM is used to name and catergorize each of these sections. Using HTML, we have agreed on the naming of each section and agreed on how to tag each section. Examples of some of these tags are h2 or div amongst other HTML "tags" When you create this document, you would want to style each section differently. Perhaps the heading should be larger or a different colour , you say. This is what CSS and Javascript do with webpages. They manipulate each section in your HTML.

EXPLAIN THE DIFFERENCE BETWEEN ACCESSING DATA FROM ARRAYS AND OBJECTS


An object is in essence a thing with properties, in addition an array is a collection of objects. Say if you wanted to create your car in code you would need to include things like top speed, accelaration, tyre width and number of seats. This could be represented as an object, for example you might decide later on you want to give the car wing doors your object could have such functions/capability e.g open_doors_vertically. If you wanted to use a whole lot cars in a race you would use an array of cars. Arrays are good for storing sequences where the order is important such as different horses and what gate they might come out of in a horse race.


EXPLAIN WHAT FUNCTIONS ARE AND WHY THEY ARE HELPFUL.


A function is essentailly an agreement to perform a task or a series of tasks whenever you are 'called' upon. The tasks will remain the same, but based on the input the result may differ. That input is a value that is provided to the function when the function is being called. They are helpful because Functions allow you to define a block of code, catergorize it by name and then execute it as many times as you wish.
